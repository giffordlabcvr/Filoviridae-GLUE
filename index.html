Mononegavirales-EVE

A GLUE project for endogenous mononegaviral elements.

Download .zip Download .tar.gz View on GitHub

Description

Mononegavirales-EVE is an open source virus sequence data resource focussed on endogenous mononegaviral elements (EMV). It has been developed as part of the DIGS-for-EVEs project.

Mononegavirales-EVE is based on GLUE, an open source, data-centric bioinformatics environment specialised for the analysis of virus sequence data.

We have created a core project - Mononegavirales-GLUE - for analysing mononegavirus (family Mononegavirales) sequence data generally, and Mononegavirales-EVE is an extension to this project.

This GLUE project contains reference information for the EMVs, set within the broader context of mononegavirus diversity, including:

  1. EMV sequences linked to auxiliary data.
  2. Alignments of EMV and mononegavirus reference sequences arranged hierarchically by clade.
  3. Alignments of EMV lineages (i.e. distinct orthologs and duplicates).

Standard nomenclature for EMV loci

This project applies a standard nomenclature to EMV loci, following the model described for ERVs in paper cited below, except using the classifier 'EMV' rather than ERV.

Gifford RJ, Blomberg B, Coffin JM, Fan H, Heidmann T, Mayer J, Stoye J, Tristem M, and WE Johnson (2018) Nomenclature for Endogenous Retrovirus (ERV) Loci Retrovirology (in press)
