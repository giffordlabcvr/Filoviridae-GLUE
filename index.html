Filoviridae-EVE

A GLUE project for endogenous filoviral elements.

Download .zip Download .tar.gz View on GitHub

Description

Filoviridae-EVE is an open source virus sequence data resource focussed on endogenous filoviral elements (EVEs). It has been developed as part of the DIGS-for-EVEs project.

Filoviridae-EVE is based on GLUE, an open source, data-centric bioinformatics environment specialised for the analysis of virus sequence data.

We have created a core project - Filoviridae-GLUE - for analysing filovirus (family Filoviridae) sequence data generally, and Filoviridae-EVE is an extension to this project.

This GLUE project contains reference information for EVEs derived from filoviruses and filovirus-like viruses:

  1. EVE sequences linked to auxiliary data.
  2. Alignments of EVE and filovirus reference sequences arranged hierarchically by clade.

Standard nomenclature for EVE loci

This project applies a standard nomenclature to EVE loci, following the model described for ERVs in the paper cited below, except using the classifier 'EVE' rather than ERV.

Gifford RJ, Blomberg B, Coffin JM, Fan H, Heidmann T, Mayer J, Stoye J, Tristem M, and WE Johnson (2018) Nomenclature for Endogenous Retrovirus (ERV) Loci Retrovirology

Installation

You can install Filoviridae-EVE on computers running Windows, MacOSX or Linux, as follows:

  1. Install GLUE, based on the GLUE installation instructions.
  2. Clone the Filoviridae-GLUE repository into your `gluetools/projects` directory.
  3. Navigate to the `gluetools/projects/Filoviridae-GLUE` directory on the command line, initiate GLUE, and build the project by issuing the following command in the GLUE shell:
Mode path: /
GLUE> run file filoviridaeProject.glue
  1. Clone the Filoviridae-EVE repository into your `gluetools/projects` directory.
  2. Navigate to the `gluetools/projects/Filoviridae-EVE` directory on the command line, initiate GLUE, and build the EVE project extension by issuing the following command in the GLUE shell:
Mode path: /
GLUE> run file Filoviridae-eve.glue
