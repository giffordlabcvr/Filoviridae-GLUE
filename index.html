Mononegavirales-EVE

A GLUE project for endogenous mononegaviral elements.

Download .zip Download .tar.gz View on GitHub

Description

Mononegavirales-EVE is an open source virus sequence data resource focussed on endogenous mononegaviral elements (EMVs). It has been developed as part of the DIGS-for-EVEs project.

Mononegavirales-EVE is based on GLUE, an open source, data-centric bioinformatics environment specialised for the analysis of virus sequence data.

We have created a core project - Mononegavirales-GLUE - for analysing circovirus (family Mononegavirales) sequence data generally, and Mononegavirales-EVE is an extension to this project.

This GLUE project contains reference information for the EMVs, including:

  1. EMV sequences linked to auxiliary data.
  2. Alignments of EMV and circovirus reference sequences arranged hierarchically by clade.
  3. Alignments of EMV lineages (i.e. distinct orthologs and duplicates).

Standard nomenclature for EMV loci

This project applies a standard nomenclature to EMV loci, following the model described for ERVs in the paper cited below, except using the classifier 'EMV' rather than ERV.

Gifford RJ, Blomberg B, Coffin JM, Fan H, Heidmann T, Mayer J, Stoye J, Tristem M, and WE Johnson (2018) Nomenclature for Endogenous Retrovirus (ERV) Loci Retrovirology

Installation

You can install Mononegavirales-EVE on computers running Windows, MacOSX or Linux, as follows:

  1. Install GLUE, based on the GLUE installation instructions.
  2. Clone the Mononegavirales-GLUE repository into your `gluetools/projects` directory.
  3. Navigate to the `gluetools/projects/Mononegavirales-GLUE` directory on the command line, initiate GLUE, and build the project by issuing the following command in the GLUE shell:
Mode path: /
GLUE> run file mononegavirales.glue
  1. Clone the Mononegavirales-EVE repository into your `gluetools/projects` directory.
  2. Navigate to the `gluetools/projects/Mononegavirales-EVE` directory on the command line, initiate GLUE, and build the EVE project extension by issuing the following command in the GLUE shell:
Mode path: /
GLUE> run file mononegavirales-eve.glue
