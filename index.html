Filoviridae-EVE

A sequence data resource for filoviruses.

Download .zip Download .tar.gz View on GitHub

Description

Filoviridae-EVE is a sequence data resource for filoviruses (family Filoviridae) developed using GLUE.

This GLUE project includes aligned, annotated genome sequence data for:

  1. Filoviruses
  2. Endogenous viral elements (EVEs) derived from filoviruses

Alignments

This project contains several distinct categories of multiple sequence alignment (MSA), each representing a distinct taxonomic level.

  1. Tip (i): Virus species (genome-length)
  2. Tip (ii): EVE orthologs (single gene)
  3. Internal: Virus genera (genome-length)
  4. Root: Viruses and EVEs (single gene)

Sequence-associated data

This project incorporates linked auxiliary data in tabular format for both virus and EVE sequences.
